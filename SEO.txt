1 - Créer du bon contenu
    -400 mots par page minimum
    -Garder un contenu à jour
    -Ne pas répéter toujours les même choses
    -Ne pas faire une page d'intro avant le Site
    -Pas de lien "cliquer ici" ou "en savoir plus"
    -Pensez à être local(préciser ville, quartier, région) Mettre .fr si votre site n'est destiné qu'à la France.

2 - HTML "seo friendly"
    -Voir html helium
    -Tester OpenGraph

3 - Site rapide (performant)
    Le robot teste le temps de chargement
    Il faut que le javascript soit lu en dernier
    Il faut faire apparaître le plus vite possible une couleur de fond
    Privilégier une image sur CSS (et non pas la balise img)

Tester son SEO : plugin "Lighthouse"
